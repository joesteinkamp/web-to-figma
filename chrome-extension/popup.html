What this does

Captures any web page and converts it into an editable Figma design — directly from your browser.

How it works

  1. You click Capture to Figma on any page
  2. The extension calls Claude Code via a local native host
  3. Claude Code calls the Figma MCP server to start a capture session
  4. A Figma overlay appears on the page — select what to capture
  5. Figma processes the capture and creates an editable design file

Requirements

  1. Claude Code — the CLI must be installed and available as claude in your terminal. Install Claude Code →
  2. Figma MCP server — must be added to your Claude Code config. Add this to ~/.claude/settings.json:
    { "mcpServers": { "figma": { "command": "npx", "args": ["-y", "figma-developer-mcp", "--stdio"] } } }

    Then restart Claude Code and follow the Figma auth prompts.

  3. Install script — browser extensions can't talk to local programs directly. The install script registers a native messaging host that lets the extension communicate with Claude Code on your machine.
