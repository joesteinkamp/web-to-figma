What this does

Captures any web page and converts it into an editable Figma design — directly from your browser.

How it works

  1. You click Capture to Figma on any page
  2. The extension calls your AI coding tool (Claude Code or Codex) via a local native host
  3. The AI tool calls the Figma MCP server to start a capture session
  4. A Figma overlay appears on the page — select what to capture
  5. Figma processes the capture and creates an editable design file

Requirements

  1. AI coding tool — install at least one:
    • Claude Code — claude CLI. Install →
    • Codex — codex CLI. Install →
  2. Figma MCP server — must be configured in your AI tool.
    Claude Code Codex

    Add to ~/.claude/settings.json or ~/.claude.json:

    { "mcpServers": { "figma": { "command": "npx", "args": ["-y", "figma-developer-mcp", "--stdio"] } } }

    Add to ~/.codex/config.toml:

    [mcp_servers.figma] command = "npx" args = ["-y", "figma-developer-mcp", "--stdio"]

    Then restart your AI tool and follow the Figma auth prompts.

  3. Install script — browser extensions can't talk to local programs directly. The install script registers a native messaging host with your browser and starts a lightweight background service that keeps your AI tool ready to handle captures. This one-time setup lets the extension communicate with the AI tool on your machine.
